## Getting Started

### Prerequisites
* NET 5 or NET Framework 4.7.2<br>
* System.Text.Json

### Installation

Clone the repo
```sh
git clone https://github.com/iwan0129/WarframeAPI.git
```

Include the built dll in your project.<br>
